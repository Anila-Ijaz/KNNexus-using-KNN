# KNNexus-using-KNN

This project implements a K-Nearest Neighbors (KNN) classifier from scratch and compares its performance with scikit-learn models on multiple datasets.
The evaluation includes accuracy, runtime analysis, and confusion matrices to assess model performance.

# Key Features:
- Custom implementation of KNN classifier
- Comparison with scikit-learn KNN and another classifier (e.g., Decision Tree)
- Experiments on datasets like Iris and Breast Cancer
- Performance evaluation using accuracy and runtime
- Visualization using confusion matrices

# Technologies:
Python, NumPy, scikit-learn, Matplotlib
